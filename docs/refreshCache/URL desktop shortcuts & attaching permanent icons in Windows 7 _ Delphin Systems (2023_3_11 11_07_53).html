URL desktop shortcuts & attaching permanent icons in Windows 7

April 23, 2012 by Tek

0

For years I’ve used shortcuts to my favourite websites on the desktop and integrated Stardock’s “Fence’s Pro” to keep my desktop shortcuts ordered and easily accessible, but often the icons particularly for my internet shortcuts would often vanish which was a constant source of irritation. In this post I intend to inform you how to not only create those Internet shortcuts but compile the relevant icons into a dynamic link library for convenience.

Going slightly off subject for a moment, another annoying issue I found relates to all my various USB connected drive icons suddenly disappearing for which I have yet to a find a permanent fix for as yet. However I will be covering my fix for this albeit pretty basic in another post as soon as possible. Generally speaking if you use a cleaner program such as WinASO registry cleaner, Ccleaner etc, it really is best to check in the options for a way of preventing certain shortcuts from being deleted while the program performs it’s cleanup procedure. I noticed my drive icon shortcuts tend to disappear when Windows own cleanup procedure takes place so always refuse when Windows asks.

Ok, back to the original subject.

Creating a URL shortcut is pretty easy. This method involves using Firefox rather than Internet Explorer as IE method is self-explanatory and besides, does anyone use IE anymore? Just kidding, I guess many people do but many also don’t.

1 – Create a folder on your desktop and name it Mywebcuts or whatever you like.

2 – Open Firefox and re-size window so you can see you’re desktop

3 – Type in the web address or click on a bookmark of your choice to access the site you wish to create a shortcut to.

4 – To the left of the web address in the address bar there may or may not be an icon. Regardless of whether there is or not, left click in this area, hold the mouse button down and drag into the previously created folder

5 – Repeat step 4 for every web page shortcut you want to create.

6 – When you have finished you will now have a folder full of web page shortcuts.

Seems odd really doesn’t it, why on earth would I ask you to create a folder for these shortcuts?

If you open the folder, you’ll notice that most if not all the Icons is the default orange Firefox icon; however some may have copied the particular websites icon which is known as a favicon. Sadly not all sites have a favicon which is a tiny 16 x 16 pixel icon that represents that particular site. Annoyingly once the browser cache has been cleared even those icon images will revert to the aforementioned Firefox icon, therefore we need a method to save the associated favicons and create some new ones for those sites that don’t have one in the first place.

A little tip here for you – For those sites that do have a personalized icon (favicon) you can save them as a .png, .jpg , .bmp and this is how you do it.

Highlight the relevant site in the bookmarks menu or toolbar and right click on it once, this opens a sub menu. Scroll down the sub menu and select Export Favicon and save it in another new folder for tidiness sake. Repeat this with every site that has its own favicon.

Ok let’s go back to the original folder we created that contains all of the Internet shortcuts. Note all the shortcuts have the .URL extension. To prevent any problems, please associate .URL with Firefox. If you are not sure how to do this I’ll explain now.

1 – Go to Windows 7 Start Menu and click on default programs.

2 – Select Associate a file type or protocol with a specific program and wait for the list of file types to appear.

3 – Scroll down the list until you find .URL then highlight and click on change program.

4 – If the program (Firefox) is not listed in the open with dialog just click on the browse button and browse to firefox.exe which by default is installed at C:\program files(x86)\Mozilla Firefox\firefox.exe

5 – Select firefox.exe and click on the OK button. Firefox is now associated with .URL extension.

If you didn’t do the above steps in a lot of cases the shortcuts would merely throw up an error message because of the lack of association between .URL and a relevant program to open it. I think if memory serves, the default association is Internet browser which always seems to throw up an error message hence the importance of a definite association.

The next step is to convert all those previously extracted favicons into proper .ICO files. There are a few ways of doing this which doesn’t require third party software but for speed I prefer to use the full freeware software Imagicon which is so simple to use. You merely drag and drop the image onto the programs window and this in turn generates the .ICO version you need.

At this juncture, I suggest you visit those sites that don’t have a favicon and find a relevant graphic that can be used as an icon for that site. Re-size to around 160 x 160 using your image editing suite and save it as a .png or if your software can save it in .ICO format, even better.

By now you should have a folder containing a number of icon image files .ICO and a folder containing your Internet URL shortcuts.

What you could do is leave all these individual .ICO files in this folder and put it somewhere else on your hard drive naturally keeping a backup or you could compile them all into a Dynamic Link Library (DLL) which is my preference. This is easily achievable by using a shareware program called Gconvert Download the full working for 14 days trial version here

Gconvert is an excellent program and only around $25 at time of writing, however; it isn’t necessary to purchase unless you know you’ll be using it a lot. After you have downloaded and installed Gconvert 5 please create a zip file of all those .ICO files and follow the steps below.

1 – Open Gconvert and click on open, browse to the relevant folder containing you’re now zipped .ICO files.

2 – Click on the zipped file and all the icons will appear in Gconverts window.

3 – On the Library menu scroll down and select – Compile Library to a DLL file (32 bit)

Saving your Icons into a dynamic link library file

4 – Name the DLL something like favsites.dll and click save.

5 – Congratulations, you’ve just created a customized .DLL file containing your internet    shortcut icons.

6 – Save the newly created DLL to a safe location away from the desktop and    remember to keep a backup too.

Back to the folder that contains your Internet shortcuts.

If you right click on any of those Internet shortcuts, select properties you’ll notice no   matter what you do you can’t find the change icon dialog. Annoying isn’t it! Therefore as ridiculous as this may seem what we need to do is create a shortcut to each of the Internet shortcuts, but before we do this, move the folder to a safe location away from the desktop.

1 – Create a shortcut to each one of the website shortcuts

2 – Right click on the first shortcut and select properties

3 – Excellent the change icon button is available so click on that and then browse to the location you saved your .DLL file to and select it accordingly. Select the correct image for the URL in question and apply. Repeat this step for each shortcut.

This concludes this post which may seem a little long winded to do what I believe should be a straight forward task but I assure you it is worth the effort as it works. However if you have a better, different or even easier way of doing this, don’t be shy share.
